Important Algebraic Formulas

  1. (a + b)2 = a2 + b2 + 2ab

  2. (a - b)2 = a2 + b2 - 2ab

  3. a2 - b2 = (a + b)(a - b)

  4. a2 + b2 = (a + b)2 - 2ab

  5. a2 + b2 = (a - b)2 + 2ab

  6. a3 + b3 = (a + b)(a2 - ab + b2)

  7. a3 - b3 = (a - b)(a2 + ab + b2)

  8. 2(a2 + b2) = (a + b)2 + (a - b)2

  9. (a + b)2 - (a - b)2 = 4ab

  10. a4 + b4 = (a + b)(a - b)[(a + b)2 - 2ab]

  11. (a - b)2 = (a + b)2 - 4ab

  12. (a + b)2 = (a - b)2 + 4ab

  13. a4 + b4 = [(a + b)2 - 2ab]2 - 2(ab)2

  14. (a + b + c)2 = a2 + b2 + c2 + 2ab + 2bc + 2ca

  15. (a + b - c)2 = a2 + b2 + c2 + 2ab - 2bc - 2ca

  16. (a - b - c)2 = a2 + b2
